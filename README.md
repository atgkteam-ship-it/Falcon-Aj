# Falcon AI

Falcon AI is a modern multi-model AI chatbot app inspired by ChatGPT.

## Features
- Multiple AI models
- Gemini API integration
- ChatGPT-like white UI
- File & image attachments
- Voice input support
- Firebase Authentication
- React Native Expo frontend
- Node.js backend

## Tech Stack
### Frontend
- React Native
- Expo
- React Navigation

### Backend
- Node.js
- Express
- Gemini API

### Database/Auth
- Firebase
